# SANTA 산타
> 등산 소셜게더링 서비스
> 
등산을 하고 싶지만, 혼자는 하기 싫을 때 산타를 이용해보세요. 
산타는 100대 명산을 등산하고 싶은 사람들이 모임을 만들어 동행을 할 수 있게 해줍니다.
